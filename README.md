# Homebrew tap for NotchRelay

This is the official Homebrew tap for [NotchRelay](https://github.com/henryvn27/notchrelay), a native macOS notch and menu-bar companion for OpenAI Codex.

No cask is published yet. The `notchrelay` cask will be added only after the first Developer ID signed and notarized release passes the repository's release verification gates. Until then, use the source repository's contributor build instructions.

Once the verified cask is available, its canonical installation command will be:

```sh
brew install --cask henryvn27/notchrelay/notchrelay
```
Homebrew tap for signed NotchRelay macOS releases.
