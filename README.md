# Homebrew tap for Cowlick

This is the official Homebrew tap for [Cowlick](https://github.com/henryvn27/cowlick), a native macOS notch and menu-bar companion for OpenAI Codex.

No cask is published yet. The `cowlick` cask will be added only after the first Developer ID signed and notarized release passes Cowlick's release verification gates. Until then, use the source repository's [contributor install](https://github.com/henryvn27/cowlick#contributor-install).

Once the verified cask is available, its canonical installation command will be:

```sh
brew install --cask henryvn27/cowlick/cowlick
```

Cowlick is an unofficial project and is not affiliated with or endorsed by OpenAI.
